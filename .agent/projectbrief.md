# Project Brief

This project is an API for automated **RAG pipeline evaluation**.

## Project Goals

Evaluate RAG pipelines using **reference-free methods**, currently supported:

- Correctness
- Faithfulness
- Relevancy

## Scope

- Automated RAG pipeline evaluation
- Agentic evaluation metric selection

## Key Requirements

- speed
- simplicity
- scaleability
- Agentic metric selection
